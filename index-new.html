COVID-19 Index Chart Visualization

Description:

This page shows counts of confirmed cases, deaths, and recovered cases for the COVID-19 pandemic. The data is pulled from the JHU CSSE data repository.

Use the dropdown to select a country/region and see its state/province counts, if available. If these are not available, the data for the selected country is shown. Select a state/province using the dropdown to highlight it in the charts. Hover over the chart to interactively highlight lines.
